# Library Management System

A database-driven Library Management System developed using C++ and MySQL to automate library operations.

## Features

- Add, update, delete books
- Search books
- Manage members
- Issue and return books
- Generate reports
- CRUD Operations

## Technologies Used

- C++
- MySQL
- SQL
- DBMS

## Database Structure

### Tables

1. Books
2. Members
3. Staff
4. IssuedBooks
